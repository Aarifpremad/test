Socket Events Documentation

1. roomjoin

Description: Triggered when a user joins a room.

Parameters (Sent by User):

{
    "roomid": "string", // Unique Room ID
    "betamount": "number" // Bet amount placed by the user
}

Response (Emitted to Client):

To the joining user: eventname -> playerjoin

{
    "player": { "id": "string", "username": "string", "balance": "number" },
    "roomid": "string"
}

To all users in the room: eventname -> allPlayers

{
    "players": [{ "id": "string", "username": "string" }],
    "roomid": "string"
}

Error Response: eventname -> error

{
    "message": "string" // e.g., "Insufficient balance!" or "User not found!"
}

2. gamestart

Description: Triggered when the game starts in a room.

Parameters (Sent by User/Admin):

{
    "roomid": "string",
    "noofplayer": "number",
    "players": [{ "userId": "string", "username": "string" }],
    "betamount": "number",
    "timeduration": "number"
}

Response (Emitted to All Players in Room):

{
    "message": "Game has started!",
    "status": "start",
    "noofplayer": "number",
    "players": [{ "userId": "string", "username": "string" }],
    "betamount": "number",
    "timeduration": "number"
}

Error Response:

{
    "message": "Room already exists!"
}

3. gameend

Description: Triggered when a game ends and results are processed.

Parameters (Sent by User/Admin):

{
    "roomid": "string",
    "players": [
        { "userId": "string", "username": "string", "rank": "number" }
    ]
}

Response (Emitted to All Players in Room):

{
    "message": "Game has ended!",
    "status": "end",
    "players": [
        { "userId": "string", "username": "string", "rank": "number" }
    ]
}

Error Response:

{
    "message": "Room not found!"
}
